# RK3588 

![RK3588](/notes/platform-notes/Rockchip/RK3588.png)

## Platform software:

* [ZephyrLabs/aarch64-playgrounds](https://github.com/ZephyrLabs/aarch64-playgrounds) - Basic examples that explore the general compute capabilities of the latest Arm Cortex and Mali techologies.

* [tsukumijima/librga-rockchip](https://github.com/tsukumijima/librga-rockchip) - Userspace driver for the Raster graphics accelerator on Rockchip "RGA" platform
    refer to: [airockchip/librga](https://github.com/airockchip/librga)

* [marty1885/rknn-superresolution](https://github.com/marty1885/rknn-superresolution) - superresolution model example on the Rockchip "RKNN2" NPU platform

* [marty1885/scirknn](https://github.com/marty1885/scirknn) - A code port of scikit-learn's MLP classifer/regresser to Rockchip "RKNN2" NPU platform

* [sravansenthiln1/rknn_tflite](https://github.com/sravansenthiln1/rknn_tflite) - TFLite examples conversion and porting to Rockchip "RKNN2" NPU platform

* [airockchip/rknn_model_zoo](https://github.com/airockchip/rknn_model_zoo) - Model examples for Rockchip "RKNN2" NPU platform
